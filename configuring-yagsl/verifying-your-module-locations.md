# Modül Konumlarınızı Doğrulama

Robotu YAGSL ile ilk kez yapılandırdıktan sonra, modüllerin havadayken doğru yönlere baktığından emin olmak için blok üzerinde test etmelisiniz. Robot programının düşündüğü şeyi yaptıklarından emin olmak için [modülleri değiştirmeniz](the-eight-steps.md#how-to-swap-module-configurations) gerekebilir.

## Düz sürüş

İlk test robota düz sürmesini emretmektir. Modüllerin gerçek hayatta ve Elastic, [FRC Web Components](../analytics-and-debugging/frc-web-components.md) veya [Advantage Scope](../analytics-and-debugging/advantage-scope.md) içinde böyle görünmesi gerekir.

<figure><img src="../.gitbook/assets/devilbots_cropped_swerve_orientation.png" alt=""><figcaption></figcaption></figure>

Gerçek hayatta böyle görünmüyorsa, sorunlu modülleri buna uyacak şekilde değiştirmeniz gerekir.

## Robotu döndürme

Robotunuz CCW+ (Saat Yönünün Tersi Pozitif) dönmelidir, ancak modüllerinizin düz sürdüğünü doğruladıktan sonraki doğrulama adımı için: Robot dönmeye çalışırken bunlardan herhangi birini yapıyor gibi görünüyorsa sorun yaşamazsınız.

{% hint style="warning" %}
Soldaki tasvir, öteleme ekseniniz robotunuzun başlığına bağlı olarak değiştiğinden [sekiz adımı](the-eight-steps.md) uygulamanızı gerektirecektir.
{% endhint %}
